### Exercise 4.1-2
Write pseudocode for the brute-force method of solving the maximum-subarray
problem. Your procedure should run in $\Theta(n^2)$ time.

    //brute-force
    FIND-MAXIMUM-SUBARRAY(A)
        max_sum = INT_MIN
        for i = 1 to A.length
            tmp_sum = 0
            for j = i to A.length
                tmp_sum = tmp_sum + A[j]
                if tmp_sum > max_sum
                    max_sum = tmp_sum
        return max_sum
